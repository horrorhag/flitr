void GeometryOverlay::setDepth ( double  z)
virtual

Adjust the depth the geometry is drawn at. Can be used to layer multiple overlays. In the default state positive z is towards the camera. The default depth value is set to 0.1. The transform in z is applied after the coordinate transform.

Parameters
zValue to adjust the depth of the geometry with.
